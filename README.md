# About Cynex

--------------------------------------------------------------------------------------------------------------------------------------------------------------
Cynex is a programming language im working on thats made in C. Its pretty early in development so its not fully finished yet. Cynex is planned and designed for simplicity, ease of use, and speed. (hence, why its made in C. for speed, but also for others.) Cynex is primarily inspired by Lua, which is why some syntax is similar to it, and it probably will be in long term. But its also planned to be friendly for cross language programmers, so if your already used to lua, then use local example = 123, or if your used to C++ or C# then you can use string, float, or bool.

Overall, Cynex is designed to be fast, easy to use, and hopefully fast to learn.
