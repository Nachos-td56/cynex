[Header]
Magic       -> "CYNB"        ; 4 bytes, identifies Cynex bytecode
Version     -> 0x01          ; 1 byte, bytecode version
Flags       -> 0xS0          ; 1 byte, signing flag (0xS0 = unsigned)
Code size   -> 0x0A           ; 1 byte (or 4 bytes if larger code), total bytecode length

---

[Constant Pool]
0x02 0x06 "Hello "    ; Type=string(0x02), Length=6, Data="Hello "
0x02 0x06 "World!"    ; Type=string(0x02), Length=6, Data="World!"

*Compact*: Each constant is just `[type][length][data]`, no padding.

---

[Variables]
0 -> partial          ; variable 0 stores "Hello "
1 -> partialend       ; variable 1 stores "World!"

*Compact*: variables are indexed by small integers for fast access.

---

[Bytecode]
0x01 0x00  ; LOAD_CONST 0      ; push constant 0 ("Hello ") to stack
0x02 0x00  ; STORE_VAR 0       ; pop stack and store in variable 0 (partial)
0x01 0x01  ; LOAD_CONST 1      ; push constant 1 ("World!") to stack
0x02 0x01  ; STORE_VAR 1       ; pop stack and store in variable 1 (partialend)
0x03 0x00  ; LOAD_VAR 0        ; push variable 0 onto stack
0x03 0x01  ; LOAD_VAR 1        ; push variable 1 onto stack
0x05       ; CONCAT            ; pop top 2 strings, concatenate, push result
0x04       ; PRINT             ; pop top of stack and print
0xFF       ; HALT              ; stop execution

*Compact*: Each instruction is 1 opcode byte + optional operand byte; stack-based VM.

---

[Optional Debug Info]
(none) ; reserved for future source mapping, line numbers, etc.

---

[Signature]
(none) ; flags=0xS0, unsigned; if flags=0xS1, would append cryptographic signature
